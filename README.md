# ewelink-home-assistant-mqtt-bridge

Cloud-first eWeLink MQTT bridge for Home Assistant using MQTT Discovery.
LAN support will be added later.

## Install (Home Assistant Add-on Repo)
1. Home Assistant → Settings → Add-ons → Add-on Store
2. Menu (top right) → Repositories
3. Add your GitHub repo URL
4. Install **eWeLink MQTT Bridge (Cloud-first)**

## Requirements
- MQTT broker (recommended: Mosquitto add-on)
