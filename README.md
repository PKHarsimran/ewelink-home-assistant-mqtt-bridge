# ewelink-home-assistant-mqtt-bridge

Cloud-first eWeLink MQTT bridge for Home Assistant using MQTT Discovery.
LAN support will be added later.

## Docker
- Build/run with Docker Compose.
- Application code lives in `app/`.

### Quick start
1. Copy `.env.example` to `.env` and fill in values when the app is implemented.
2. Run `docker compose up --build`.

## Requirements
- MQTT broker
